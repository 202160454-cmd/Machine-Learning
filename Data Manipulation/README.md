# Data Manipulation

Este módulo corresponde al Nivel IV dentro de la arquitectura de formación en Inteligencia Artificial.

En esta sección se trabajan técnicas fundamentales para la manipulación, limpieza y transformación de datos utilizando Python y librerías como NumPy y Pandas.

---

## Contenido

- Generación de datos aleatorios
- Manipulación de arrays con NumPy
- Creación de DataFrames con Pandas
- Cálculo de estadísticas (media, promedio)
- Organización y transformación de datos

---

## Estructura

El notebook sigue un flujo típico de análisis de datos:

- Generación o carga de datos
- Transformación de datos
- Creación de estructuras (DataFrame)
- Análisis básico
- Interpretación de resultados

## Objetivo

El objetivo de este módulo es desarrollar habilidades en el manejo de datos, desde su generación hasta su análisis, como base para modelos de machine learning.
