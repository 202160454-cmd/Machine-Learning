# Estadística Inferencial

Este módulo corresponde al Nivel II dentro de la arquitectura de formación en Inteligencia Artificial.

En esta sección se estudian los fundamentos de la inferencia estadística, permitiendo realizar estimaciones y tomar decisiones basadas en datos.

---

## Contenido

- Pruebas de hipótesis
- Estimadores (UMVUE)
- Análisis estadístico inferencial
- Interpretación de resultados

---

## Estructura

Los notebooks siguen una estructura orientada al análisis estadístico:

- Planteamiento del problema
- Desarrollo teórico
- Implementación en Python
- Análisis de resultados
- Conclusiones

---

## Archivos principales

- Solución_Pruebas_de_hipótesis.ipynb → Aplicación de pruebas de hipótesis  
- UMVUE_beta.ipynb → Estimadores y propiedades estadísticas  
- data.html → Visualización o resultados del análisis  

---

## Objetivo

El objetivo de este módulo es comprender cómo inferir información sobre una población a partir de una muestra, utilizando herramientas estadísticas.
