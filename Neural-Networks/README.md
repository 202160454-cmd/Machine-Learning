# Neural Networks

Este módulo corresponde al Nivel VI dentro de la arquitectura de formación en Inteligencia Artificial.

En esta sección se estudian los fundamentos de las redes neuronales, incluyendo su estructura, funcionamiento y aplicación en problemas de machine learning.

---

## Contenido

- Fundamentos de redes neuronales
- Perceptrón y arquitectura básica
- Propagación hacia adelante (forward propagation)
- Funciones de activación
- Implementación de redes neuronales en Python

---

## Estructura

Los notebooks siguen un enfoque práctico:

- Explicación teórica
- Implementación del modelo
- Ejecución del algoritmo
- Análisis de resultados
- Conclusiones

---

## Archivos principales

- Neuronas.ipynb → Fundamentos de redes neuronales  
- Tiny_nn.ipynb → Implementación de una red neuronal simple  
- utils.py → Funciones auxiliares para los modelos  

---

## Objetivo

Comprender cómo funcionan las redes neuronales y cómo pueden implementarse desde cero para resolver problemas de aprendizaje automático.
