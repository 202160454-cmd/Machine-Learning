# Streamlit 

Este módulo corresponde al Nivel IV dentro de la arquitectura de formación en Inteligencia Artificial.

En esta sección se desarrollan aplicaciones interactivas utilizando Streamlit, permitiendo visualizar modelos, datos y simulaciones de forma dinámica.

---

## Enfoque del módulo

El enfoque está orientado a la creación de interfaces para proyectos de data science, facilitando la comunicación de resultados y la interacción con modelos.

---

## Contenido

- Desarrollo de aplicaciones web con Streamlit  
- Visualización de datos  
- Simulación de procesos  
- Interacción con modelos  

---

## Aplicaciones

- calendar.py → Aplicación basada en fechas  
- poisson.py → Simulación de procesos Poisson  
- Markov.py → Modelado de procesos estocásticos  
- clean.py → Procesamiento de datos  
- descriptiva.py → Análisis estadístico  

---

## Estructura

Cada aplicación sigue el flujo:

- Entrada de datos  
- Procesamiento  
- Visualización  
- Interacción con el usuario  

---

## Objetivo

Transformar modelos y análisis en aplicaciones interactivas que faciliten la interpretación de resultados.

---

## Mejora aplicada

- Organización modular de aplicaciones  
- Separación de lógica por archivo  
- Mejora en presentación visual  
- Enfoque en usabilidad y experiencia del usuario
